## Анализатор логов Nginx 

Приложение позволяет анализировать логи Nginx и считать разные метрики длительности HTTP запросов и строить HTML отчеты с данными.
Работает приложение по следующему алгоритму:

- читается локальный конфиг из кода с возможностью переопределить его через опцию командной строки `--config`
- приложение анализирует логи из папки log и выбирает последний файл по дате в названии
- из данного файла собирается статистика по времени ответа на каждый URL адрес
- на основе этих данных вычисляются метрики для каждого адреса
- строится отчет на основе шаблона report.html и записывается в файл в директории `reports`

Запуск скрипта осуществляется из командной строки

```commandline
python3 log_analyzer.py
```

Возможно указать файл конфигурации с помощью опции

```commandline
python log_analyzer.py --config new.conf
```

Тесты на приложение запускаются при помощи

```commandline
python tests.py
```

В папке `reports` хранится шаблон отчета и js файл для построения итоговых отчетов.
